# Chat AngularJS
![Chat em AngularJS](http://i.imgur.com/z5rjp6A.png)

## Tecnologias
* AngularJS
* NodeJS
* HTML
* Gulp
* Bower

### NPM
[![forthebadge](http://forthebadge.com/images/badges/fuck-it-ship-it.svg)](http://forthebadge.com)
Instalando NPM dentro do diretório
`npm install`

### Bower
Instalação do Bower
`npm install -g bower`

### Gulp
Instalação do Gulp
`npm install gulp-cli -g`
`npm install gulp -D`
